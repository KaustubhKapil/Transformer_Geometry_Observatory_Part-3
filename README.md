# TGO-III: Semantic Geometry Observatory
Transformer Geometry Observatory Part-3 (TGO-III) investigates the emergence of semantic structure in Vision Transformers throughout supervised training. Building upon the spectral and representation geometry analyses of TGO-I and TGO-II, this observatory studies whether the increasing representational complexity observed during training corresponds to progressively richer semantic organization. TGO-III analyzes semantic geometry through linear probing, class separability, nearest-neighbor purity, Fisher discriminant analysis, and label alignment to determine whether representation manifold expansion reflects genuine semantic discovery or purely geometric complexity.
